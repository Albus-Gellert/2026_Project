# Global Water Stress Explorer Prototype

可运行的 Vue 3 + TypeScript + Apache ECharts 高保真仪表板原型。

```powershell
pnpm install
pnpm run dev
```

生产构建：

```powershell
pnpm run build
```

所有数值均为界面演示用模拟数据，不是 AQUASTAT 事实数据。

完整的项目状态、组件说明、数据替换方法和新手指南见上级目录的 [`README.md`](../README.md) 与 [`ARCHITECTURE.md`](../ARCHITECTURE.md)。
